## Design Guidelines

### Font and Colors

- **Font**: Garamond
- **Font Sizes**:
  - Heading: 14px
  - Subheadings: 12px
  - Body Text: 10px
  - Small Text: 8px
  - Minimized Text: 7px (use sparingly)

- **Colors**:
  - Lavender: #C3B7EA
  - Navy: #000080
  - Black: #000000

- **Background**: Dark gray tone for dark mode
